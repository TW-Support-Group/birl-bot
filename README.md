# birl-bot

## Run locally

> [!NOTE]
> In order to run the bot you'll need access to the TOKEN.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

## Deploy with Docker

This command will build a multiplatform image for both arm64 and amd64:

```bash
make build
```
